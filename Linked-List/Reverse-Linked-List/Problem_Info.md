# Problem Name: Reverse Linked List
### LeetCode Number:
- 206
### Difficulty:
- Easy
### Category:
- Linked List / Recursion

### Description:
- Given the `head` of a singly linked list, reverse the list, and return the reversed list.
- A singly linked list is a data structure where each element (node) contains a value and a reference (next) to the next node in the sequence.

### Constraints:
* The number of nodes in the list is the range `[0, 5000]`.
* `-5000 <= Node.val <= 5000`
