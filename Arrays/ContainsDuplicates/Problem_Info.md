# Problem Name: Contains Duplicate
### LeetCode Number:
- 217
### Category:
- Arrays / Hash Table

### Description:
- Given an integer array `nums`, return `true` if any value appears at least twice in the array.
- Return `false` if every element is distinct.

### Constraints:
* `1 <= nums.length <= 10⁵`
* `-10⁹ <= nums[i] <= 10⁹`
